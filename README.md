# Bank Application (Core Java)

## Project Description
This is a simple Bank Application built using Core Java. It performs basic banking operations like account creation, deposit, withdrawal, and balance check.

## Features
- Create account
- Deposit money
- Withdraw money
- Check balance

## Technologies Used
- Java
- OOP Concepts
- Collections (if used)

## Project Structure
src/main/java/org/example/model/Account.java

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run main class
